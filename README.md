# Game 01 - Platform game

## Dependencies
``` shell
$ brew install raylib cmake
$ sudo apt install mesa-utils libglu1-mesa-dev freeglut3-dev mesa-common-dev libglew-dev libglfw3-dev libglm-dev
```

## Build
``` shell
$ cmake -S . -B build
$ cmake --build build
```
or the VS Code build task

## Run
``` shell
$ build/game
```
